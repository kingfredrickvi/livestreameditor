Livestream Editor Collaberatively edit livestreams

What Is This

Livestream Editor is a cloud-based web application that allows the community to collaberatively edit the highlights for streamers with other members in the community in real time. As new highlights are added, other editors instantly see them in their list. The web application is lightweight and works on most browsers, phones and tablets.

Video Editor

How To Take Part

If you would like to take part in helping edit livestreams, click "log in" at the top and log into the website using Twitch. Once you've logged in, you should see an error saying you do not have permission. Send an email to editors@livestreameditor.com requesting that you would like access and your Twitch username, or message the right person in Discord.

How It Works

When a streamer that the website follows finishes their livestream, the website will automatically download the stream archive and add it to the website. It can take upwards of 4 hours for this processing to complete.

Once the website finishes, members of the community can view the editor page for that archive. They can play through the footage and mark start and end points. Each pair of start and end points is called a segment. After a segment is added, a moderator will review it and either mark it as a green check (approved) or a red X (rejected).

Segments List

Final Product

Once all segments have been reviewed, a moderator can export the video. This will generate an mp4 file which is called an artifact. This mp4 is all of the segments sowed together into one video. From there, the streamer can download the mp4 and immedately upload it to YouTube or edit it further.

Editor Controls

How To Use Editor Controls

The editor has several buttons to control the video and segments.

  1. The current time in seconds of the video
  2. The starting time of the segment
  3. The ending time of the segment
  4. Skip backwards 45 seconds
  5. Skip backwards 0.10 seconds
  6. Stop the video and go back to where first started playing or last place seeked
  7. Pause the video, click again to resume
  8. Play the video
  9. Skip forwards 0.10 seconds
  10. Skip forwards 45 seconds
  11. Set start of segment to the current time. This will place an orange bar on the timeline
  12. Seek to the start of the segment
  13. Seek to the end of the segment
  14. Set end of segment to the current time. This will place an orange bar on the timeline
  15. Stop when reach end of segment. Light-grey means function is on. Click button to toggle
  16. Add segment to the list on the right, seek to the end of the segment, and close the segment

Segment Controls

How To Use Editor Controls

Segments have a few buttons for control.

  1. View the segment. This will set the video's start and end segment to this segment's start and end
  2. View the segment and autoplay the video
  3. Delete the segment

In the future, there will be the ability for others to like segments to help moderator filter through segments.

Editor Controls When Viewing Segment

How To Edit A Segment

It's easy to go back and edit a segment. In the segment list, click the View or Play button. Adjust the start and end points of the segments and then click the save button (labeled 2 in the image above). A segment cannot be edited if it is marked as approved or rejected unless you are a moderator.

Timeline Controls

How To Control The Timeline

  1. (Moderators only) Generate artifact
  2. Zoom out on the timeline
  3. Zoom in on the timeline
  4. Toggle autoplaying the next approved segment (must be already viewing a segment)
  5. Toggle autoplaying the next approved segment or pending review segment (must have autoplay toggled)
